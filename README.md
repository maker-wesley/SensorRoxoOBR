- SensorRoxoOBR -

Biblioteca para facilitar o uso do sensor de cor TCS34725 integrado a um multiplexador I2C para robôs da OBR (Olimpíada Brasileira de Robótica).

Funcionalidades:
- Alternância automática de canais do multiplexador.
- Leitura e normalização de dados (RGB + Clear).
- Sistema de calibração simples para detecção de cores (Preto, Branco, Verde, Vermelho, Azul).

Instalação:
- Disponível diretamente pelo Gerenciador de Bibliotecas do Arduino.
